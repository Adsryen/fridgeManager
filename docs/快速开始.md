# 🚀 快速开始

## 一键启动（Windows）

### 方式 1：自动配置（推荐）
```bash
# 双击运行
setup.bat

# 然后双击运行
start.bat
```

### 方式 2：手动启动
```bash
# 1. 创建虚拟环境（首次运行）
python -m venv venv

# 2. 激活虚拟环境
venv\Scripts\activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 启动应用
python app.py
```

## 访问应用

打开浏览器访问：http://127.0.0.1:8080

## 首次使用

1. 点击"立即注册"创建账号
2. 填写用户名、邮箱、密码
3. 注册成功后自动登录
4. 开始管理你的冰箱！

## 数据安全

- 数据库文件：`data/fridge.db`（已加入 .gitignore）
- 密码加密：PBKDF2-SHA256 + 随机盐值
- 数据隔离：每个用户只能看到自己的数据

## 常见问题

### Q: 忘记密码怎么办？
A: 目前需要联系管理员重置，后续版本会添加邮箱找回功能

### Q: 可以导出数据吗？
A: 数据存储在 SQLite 数据库中，可以使用 SQLite 工具导出

### Q: 支持多少用户？
A: 理论支持 1 万用户，每用户 100 条数据（100 万条记录）

### Q: 云服务器配置要求？
A: 最低 512MB 内存 + 10GB 硬盘即可

## 技术栈

- **后端**: Python 3.12+ + Flask 3.1.0
- **数据库**: SQLite 3
- **前端**: Bootstrap 5 + jQuery 3.6
- **认证**: Session + PBKDF2 密码哈希
