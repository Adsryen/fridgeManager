# 快速开始

## 系统要求

- Python 3.12+
- Windows 操作系统

## 安装步骤

### 1. 克隆项目

```bash
git clone https://github.com/Adsryen/fridgeManager.git
cd fridgeManager
```

### 2. 安装依赖

```bash
py -m pip install -r requirements.txt
```

### 3. 初始化数据库

首次运行需要执行数据库迁移（如果已有旧数据库）：

```bash
py scripts/migrate_add_admin_fields.py
```

### 4. 创建管理员账号

系统已预设默认管理员账号：

```
用户名: admin
密码: admin
```

运行以下命令初始化管理员账号：

```bash
py scripts/init_admin.py
```

⚠️ **安全提示**：首次登录后请立即修改密码！

### 5. 启动服务器

#### 方式一：使用快速启动脚本（推荐）

```bash
scripts\quick_start.bat
```

#### 方式二：手动启动

```bash
py run.py
```

### 6. 访问系统

服务器启动后，在浏览器中访问：

- 本地访问: http://127.0.0.1:8080
- 局域网访问: http://192.168.0.138:8080

## 功能说明

### 普通用户功能

1. **用户注册/登录**
   - 注册新账号
   - 密码强度检测
   - 记住用户名

2. **物品管理**
   - 添加/编辑/删除物品
   - 按类别、位置筛选
   - 过期提醒
   - 搜索功能

3. **个人中心**
   - 查看/编辑个人资料
   - 修改密码
   - 用户设置

4. **用户设置**
   - 通知设置（过期提醒、提前天数）
   - 显示设置（每页数量、默认视图）
   - 隐私设置（公开资料）

### 管理员功能

1. **管理员仪表板**
   - 系统统计信息
   - 用户统计
   - 物品统计

2. **用户管理**
   - 查看所有用户
   - 查看用户详情
   - 禁用/启用用户
   - 设置/取消管理员权限
   - 删除用户


## 开发模式

启用调试模式：

```bash
set DEBUG=1
py run.py
```

## 注意事项

1. **数据安全**
   - 数据库文件 `data/fridge.db` 已被 Git 忽略
   - 请定期备份数据库文件

2. **密码安全**
   - 系统使用 PBKDF2-SHA256 加密密码
   - 建议使用强密码
   - 定期修改密码

3. **生产环境**
   - 不要在生产环境使用 Flask 开发服务器
   - 建议使用 Gunicorn 或 uWSGI
   - 配置 HTTPS

## 故障排除

### 数据库错误

如果遇到数据库字段不存在的错误，运行迁移脚本：

```bash
py scripts/migrate_add_admin_fields.py
```

### 端口被占用

修改 `run.py` 中的端口号，或设置环境变量：

```bash
set PORT=8081
py run.py
```

### Python 版本问题

确保使用 Python 3.12+：

```bash
py --version
```
