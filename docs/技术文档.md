## 数据库设计

### 用户表 (users)
```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    username VARCHAR(80) UNIQUE NOT NULL,
    email VARCHAR(120) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    is_active BOOLEAN DEFAULT TRUE,
    current_fridge_id INTEGER,
    FOREIGN KEY (current_fridge_id) REFERENCES fridges (id)
);
```

### 家庭表 (families)
```sql
CREATE TABLE families (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name VARCHAR(100) NOT NULL,
    description TEXT,
    invite_code VARCHAR(20) UNIQUE NOT NULL,
    created_by INTEGER NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (created_by) REFERENCES users (id)
);
```

### 冰箱表 (fridges)
```sql
CREATE TABLE fridges (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name VARCHAR(100) NOT NULL,
    description TEXT,
    owner_id INTEGER NOT NULL,
    family_id INTEGER,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (owner_id) REFERENCES users (id),
    FOREIGN KEY (family_id) REFERENCES families (id)
);
```

### 物品表 (items)
```sql
CREATE TABLE items (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name VARCHAR(100) NOT NULL,
    quantity INTEGER DEFAULT 1,
    expire_date DATE,
    location VARCHAR(20) NOT NULL,
    category VARCHAR(20) NOT NULL,
    fridge_id INTEGER NOT NULL,
    created_by INTEGER NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (fridge_id) REFERENCES fridges (id),
    FOREIGN KEY (created_by) REFERENCES users (id)
);
```

### 用户设置表 (user_settings)
```sql
CREATE TABLE user_settings (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER UNIQUE NOT NULL,
    theme VARCHAR(20) DEFAULT 'light',
    language VARCHAR(10) DEFAULT 'zh-CN',
    notification_enabled BOOLEAN DEFAULT TRUE,
    expire_reminder_days INTEGER DEFAULT 3,
    ai_model VARCHAR(50) DEFAULT 'gpt-3.5-turbo',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users (id)
);
```

## 安全机制

### 认证与授权
- **JWT Token**：用于用户身份验证
- **Token 刷新**：自动刷新过期令牌
- **权限控制**：基于角色的访问控制
- **密码加密**：使用 Werkzeug 进行密码哈希

### 数据安全
- **SQL 注入防护**：使用参数化查询
- **XSS 防护**：前端输入验证和转义
- **CSRF 防护**：使用 CSRF 令牌
- **CORS 配置**：限制跨域访问

### 隐私保护
- **数据隔离**：用户数据完全隔离
- **敏感信息**：不在前端存储敏感数据
- **日志脱敏**：日志中不记录敏感信息
